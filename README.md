# gulp 压缩功能

##1. 首先全局下一定要有安装gulp
npm install -g gulp-cli

##2. 在当前文件夹下执行
npm install -D --save gulp

##3. 把对应要压缩的css js html img 放到对应的文件夹

##4. 执行 gulp
或者对应只执行：
gulp bulid:img
gulp bulid:js
gulp bulid:css
gulp bulid:html

##5. 在对应的文件夹找到输出文件

##6. 图片只支持PNG, JPEG, GIF and SVG 

##7. 如果进来发现没有node_modules文件夹 请执行
   npm install 命令
